Программа для поиска сетевых принтеров готова! Вот её основные возможности:
Функции программы:

Автоматическое определение локальной сети
Поиск активных устройств через ping
Сканирование типичных портов принтеров (9100, 515, 631, 80, 443, 161)
Определение типа принтера по открытым портам
Получение hostname устройств
Многопоточное сканирование для ускорения работы

Как использовать:

Интерактивный режим:

bashpython printer_scanner.py

Командная строка с указанием сети:

bashpython printer_scanner.py 192.168.1.0/24
Что программа находит:

Принтеры HP с поддержкой JetDirect (порт 9100)
Принтеры с поддержкой LPD/LPR (порт 515)
Принтеры с поддержкой IPP/CUPS (порт 631)
Устройства с web-интерфейсом (порты 80/443)
Устройства с SNMP (порт 161)

Результат включает:

IP адрес принтера
Имя хоста (если доступно)
Тип устройства
Список открытых портов с описанием

Программа работает на Windows, Linux и macOS без дополнительных зависимостей, используя только стандартную библиотеку Python.
